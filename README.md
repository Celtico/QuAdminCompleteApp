- QuCKEditor
- QuPHPMailer
- QuAdminDemo
- QuWebDemo

Installation
========================
- Download and unzip
- Load the database folder /db
- Configure the connection to the database in /config/autoload/global.php
- Create a virtual host in apache, that points to the folder /public
